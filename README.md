### Windows / Vim

Use vim and deoplete.nvim on Windows.

#### how to use.

```markdown
# Use scoop.
scoop install vim

# Default, folder directroy
mkdir GitHub

# Move.
cd GitHub

# Clone.
git glone git@github.com:takkii/winvim.git

# move.
cd winvim

# Create, .vimrc and .gvimrc
ruby win_install.rb

# Delete, .vimrc and .gvimrc
ruby delete.rb
```
